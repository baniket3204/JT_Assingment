How to ?

1)Enter your number one by one in the input feild and click on the "Add Number" button simultaneously.
2)Press the button "Result" to display result on the console.
3)Note : Console can accessed by right clicking on the browser screen.

Outcomes

Time Complexity : O(N)
Space Complexity : O(1)

Test Cases

1)Input - [ 2, 1, 5, 3, 11, 7 ] Output - [ 1, 5, 2, 11, 3, 7 ]
2)Input - [ 456 ,789 , 900 , 1500] Output - [ 456, 900, 789, 1500 ]
